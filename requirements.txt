# ====================================================================
# PLATFORM-SPECIFIC NOTES
# ====================================================================

# Windows:
#   - May need Microsoft C++ Build Tools for lxml
#   - Download from: https://visualstudio.microsoft.com/downloads/
#   - Or install pre-built wheels: pip install --only-binary :all: lxml

# Linux:
#   - May need: sudo apt-get install python3-dev libxml2-dev libxslt1-dev
#   - For Ubuntu/Debian specifically

# macOS:
#   - Xcode command line tools may be required: xcode-select --install
#   - lxml usually works out of the box with pip

# ====================================================================
# OLLAMA INSTALLATION
# ====================================================================

# Ollama is NOT a Python package installed via pip
# Install separately:
#
#   Windows: https://ollama.ai/download
#   Linux:   curl -fsSL https://ollama.ai/install.sh | sh
#   macOS:   brew install ollama
#
# Then download model:
#   ollama pull llama3.2:3b

# TROUBLESHOOTING

#   1. Update pip: pip install --upgrade pip
#   2. Use specific versions: pip install requests==2.31.0
#   3. Try with --user flag: pip install --user requests
#   4. Check Python version: python --version (must be 3.8+)
